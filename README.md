# 优选商城 - 电商平台

一个现代化的电商平台前端页面，具有完整的商品展示、购物车功能和响应式设计。

## 功能特点

### 🛍️ 商品展示
- 分类浏览：数码电子、服装配饰、家居用品、图书文具、运动户外
- 商品搜索：实时搜索过滤商品
- 视图切换：网格视图和列表视图
- 商品详情：价格、评分、评论数展示

### 🛒 购物车功能
- 添加商品到购物车
- 实时更新商品数量
- 删除购物车商品
- 计算总价
- 侧边栏购物车展示

### 📱 用户体验
- 现代化UI设计
- 响应式布局（适配移动端、平板、桌面）
- 流畅的动画效果
- 直观的操作界面

### 🎨 设计特色
- 渐变色导航栏
- 卡片式商品展示
- 轮播图促销区域
- 星级评分系统
- 悬停动画效果

## 技术栈

- **HTML5**: 语义化标签，现代化结构
- **CSS3**: Flexbox/Grid布局，动画效果，响应式设计
- **JavaScript**: ES6+语法，DOM操作，事件处理
- **Font Awesome**: 图标库

## 使用方法

1. 下载或克隆项目文件
2. 直接在浏览器中打开 `index.html` 文件
3. 开始使用电商平台功能

## 主要功能演示

### 商品浏览
- 点击顶部分类标签筛选商品
- 使用搜索框搜索特定商品
- 切换网格/列表视图模式
- 点击商品卡片查看详情

### 购物车操作
- 点击"加入购物车"按钮添加商品
- 点击右上角购物车图标查看购物车
- 在侧边栏中调整商品数量
- 删除不需要的商品

### 响应式体验
- 在移动设备上自动适配布局
- 触摸友好的交互设计
- 优化的移动端购物体验

## 开发说明

本项目采用纯前端技术实现，无需后端支持即可运行。所有商品数据通过JavaScript模拟，购物车数据存储在内存中。

如需部署到生产环境，建议：
1. 添加真实的后端API
2. 实现用户认证系统
3. 添加数据库支持
4. 优化性能和SEO
